# Minions Scaffolds

Project scaffolding templates, tech stack configs, and initial file sets

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-scaffolds/sdk`) — Core library for Node.js
- **Python SDK** (`minions-scaffolds`) — Python bindings
- **CLI Tool** (`@minions-scaffolds/cli`) — Command-line interface
- **Web App** — Interactive playground at [scaffolds.minions.wtf](https://scaffolds.minions.wtf)
