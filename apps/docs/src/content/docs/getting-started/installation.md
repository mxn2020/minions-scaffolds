---
title: Installation
description: Install Minions Scaffolds packages
---

## TypeScript / Node.js

```bash
npm install @minions-scaffolds/sdk minions-sdk
```

## Python

```bash
pip install minions-scaffolds
```

## CLI

```bash
npm install -g @minions-scaffolds/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
